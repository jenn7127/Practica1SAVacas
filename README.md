
# SA

## Inicio

Instalar las dependecias (librerias)
```bash
npm install
```

Iniciar el server
```bash
npm start
# http://localhost:3000/
```


**Importante**

Si miran que alguien agrego cambios al package.json

```
npm install
```


## Pruebas Unitarias

Se utilizara `mocha`, este se invoca con

```bash
npm run test
```
